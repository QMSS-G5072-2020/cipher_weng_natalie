Installation¶

Stable release¶

To install cipher_yjw2106, run this command in your terminal:

$ pip install -u cipher_yjw2106

This is the preferred method to install cipher_yjw2106, as it will always install the most recent stable release.

If you don't have pip installed, this Python installation guide can guide you through the process.

From sources¶

The sources for cipher_yjw2106 can be downloaded from the Github repo.

You can either clone the public repository:

$ git clone git://github.com/yjw224/cipher_yjw2106

Or download the tarball:

$ curl  -OL https://github.com/yjw224/cipher_yjw2106/tarball/main

Once you have a copy of the source, you can install it. The method of installation will depend on the packaging library being used.

For example, if setuptools is being used (a setup.py file is present), install cipher_yjw2106 with:

$ python setup.py install

If poetry is being used (poetry.lock and pyproject.toml files are present), install cipher_yjw2106 with:

$ poetry install
